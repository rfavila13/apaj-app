# APAJ - Sistema de Apoio à Ludopatia

Sistema desenvolvido para a Associação de Prevenção e Apoio ao Jogo (APAJ).

## 🌱 Sobre o Projeto

Este sistema oferece:

### Para Pacientes:
- Contador de dias sem jogar
- Registro de recaídas e economia
- Auto-teste de avaliação
- Dicas de prevenção
- Solicitação de ajuda profissional

### Para Psicólogos:
- Dashboard com visão geral
- Gestão de pacientes
- Registro de sessões
- Gestão de grupos terapêuticos
- Atendimento de solicitações

## 🚀 Como Rodar Localmente

1. Instale as dependências:
```bash
npm install
```

2. Rode o projeto:
```bash
npm run dev
```

3. Acesse: http://localhost:3000

## 📦 Deploy na Vercel

1. Faça push do código para o GitHub
2. Acesse [vercel.com](https://vercel.com)
3. Importe o repositório
4. Deploy automático!
